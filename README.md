# mobilender-test-app
This project is a little test to exemplify like works:
* Vue
* Vuex
* Axios
* Reusable web components
* Jest

# Clone Project
https://github.com/montoyaguzman/mobilender-test-app.git

## Project setup
```
npm install
```

### Compiles and serve
```
npm run serve
```
### Execute test
```
npm run test:unit
```
